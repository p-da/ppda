# Loader Page YPPDA

**Yayasan Pondok Pesantren Darul Abror**

Halaman ini berfungsi sebagai **loader atau pengalih (redirect page)** yang secara otomatis mengarahkan pengunjung ke website utama Yayasan Pondok Pesantren Darul Abror.

Loader page dibuat menggunakan **HTML sederhana** agar ringan, cepat dimuat, dan dapat digunakan pada berbagai layanan hosting statis seperti GitHub Pages atau platform deploy lainnya.

Tujuan halaman ini adalah untuk mempermudah pengelolaan akses menuju website utama, serta dapat digunakan sebagai halaman penghubung sebelum pengguna masuk ke sistem atau aplikasi yang tersedia di lingkungan Yayasan Pondok Pesantren Darul Abror.
